class modular_trader.logging.TradingLogger(logger: ~loguru._logger.Logger = <loguru.logger handlers=[(id=0, level=10, sink=<stderr>)]>, file_path: ~os.PathLike = PosixPath('/home/runner/.modular_trader_log/trader.log'), rotation_size_mb: int = 100, retention_days: int = None)#

Bases: BaseLogger

A logger that logs messages to a file.

This logger logs messages to a file. The file will be rotated when it reaches a certain size, and old files will be deleted after a certain number of days.

logger#

The underlying logger.

Type:

Logger

file_path#

The path to the log file.

Type:

os.PathLike

rotation_size_mb#

The maximum size of the log file before it is rotated. If None, the file will not be rotated.

Type:

int

retention_days#

The number of days to keep old log files. If None, old files will not be deleted.

int
